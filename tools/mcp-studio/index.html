operational_safety_boundaries.md

        

      

      
    

    
    
    

      

        

          🧠 WHAT THE AGENT CAN DO
        

        

          Read metrics, scan directories, validate configurations, build deployment templates, and diagnose failures based on structured logs.
        

      


      

        

          ⚠️ WHAT THE AGENT SHOULD NOT DO
        

        

          Delete master databases, push force branch changes, modify IAM root permission credentials, or clean production backup clusters without manual approval gates.
        

      


      

        

          🛡️ HUMAN-IN-THE-LOOP APPROVAL WORKFLOWS
        

        

          Implement webhook locks or proxy checks requiring developers to run approval tasks via messaging channels before dangerous write commands execute.
        

      


      

        

          📝 AUDIT LOGGING STANDARDS
        

        

          Stream syslog agent payloads (prompt strings, tool called, response arrays) to Elasticsearch or syslog endpoints for compliance checks.
